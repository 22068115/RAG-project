This subject is a practical introduction to coding and problem-solving in data science using the R programming language. You'll learn to develop and analyse R scripts to perform simple data analysis with diverse data sets. The subject also teaches you to create professional, reproducible reports with visualisations and introduces you to industry-standard tools like Linux and Git. By the end, you'll have a strong foundation in data, technology, and computer science.

Upon completion of this subject, graduates will be able to:
1.
Write R programs to conduct Data Science tasks.

keyboard_arrow_down
2.
Debug programs to fix R scripts that perform data science and analytics tasks.

keyboard_arrow_down
3.
Create basic data visualisations and reproducible reports using R graphics, Markdown, and R Markdown to support decision making.

keyboard_arrow_down
4.
Integrate large data sets to enable complex data analysis.

keyboard_arrow_down
5.
Manage script development through version control and testing in the Git environment with the help of Unix commands.
